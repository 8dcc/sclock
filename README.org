* Keybinds

| Key    | Description                         |
|--------+-------------------------------------|
| ~q~, ~ESC~ | Quit the program                    |
| ~t~, ~p~   | Change color palette (theme)        |
| ~g~      | Toggle the grid (dotted background) |
| ~f~, ~F11~ | Toggle full-screen                  |
